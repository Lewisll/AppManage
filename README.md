# AppManage
一个管理手机应用的工具App，不仅可以方便的查看手机应用的属性信息，如：基本信息（ICON、图标、包名、签名、版本信息、安装时间等）、存储信息、权限信息。还可以直接运行和卸载应用，同时提供跳转到系统的应用设置来操作应用。
